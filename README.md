## ✨ 核心特性

- **🚀 全项目上下文理解** —— AI 真正理解整个代码库，而非单文件，支持几乎所有的文档理解，包括小说创作、文案编写等
- **💬 自然语言编程** —— 用中文/英文直接说需求，AI 自动实现
- **🔄 一键重构与优化** —— 代码优化、性能提升、架构调整
- **🌐 支持 100+ 种编程语言**（重点优化 Python、TypeScript、Go、Rust、Java 等）
